# Astroweb

Sistema de Gestión de Fotografías para Mercaderistas

## Descripción

Astroweb es una aplicación web diseñada para la gestión y administración de fotografías tomadas por mercaderistas en campo. El sistema permite cargar, organizar y revisar imágenes de manera eficiente.

## Características

- Gestión centralizada de fotografías
- Interfaz intuitiva y responsive
- Almacenamiento seguro de imágenes
- Sistema de categorización y búsqueda

## Tecnologías Utilizadas

### Backend
- **Python** - Lenguaje de programación principal
- **Flask** - Framework web minimalista
- **SQL Server** - Base de datos relacional

### Frontend
- **Vue.js** - Framework JavaScript progresivo
- **JavaScript Vanilla** - Funcionalidades complementarias

### Infraestructura
- **Docker** - Contenerización y despliegue

## Instalación

### Prerrequisitos
- Python 3.8+
- SQL Server
- Docker (opcional)

### Instalación Web

1. Clonar el repositorio
2. Crear un entorno virtual (recomendado)
3. Instalar dependencias:
   
   pip install -r requirements.txt
   python run.py
